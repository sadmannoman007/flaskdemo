About this app

This small Flask application is a demo that searches Wikipedia and displays the result using Jinja templates. It was built to practise:

  • Using Flask routes and view functions
  • Working with HTML templates and template inheritance
  • Handling form input and POST requests
  • Using sessions to remember the last search term
  • Calling the Wikipedia API from Python code

Use the navigation links at the top of the page to return home or try a new search.
